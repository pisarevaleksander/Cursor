# Cursor Mobile Preview

## Интерактивный Canvas-артефакт (без терминала)

Открой `canvas-artifact.html` в браузере (двойной клик) — это полностью интерактивный single-file HTML:
- Todo List
- drag-and-drop между колонками To do / Doing / Done
- фильтры (All / Active / Done)
- сохранение задач в `localStorage`
- Tailwind через CDN

## Альтернатива: предыдущий мобильный артефакт

Также доступен `artifact-preview.html` (мобильный UI с таб-баром).

## Запуск React-версии (опционально)

```bash
npm install
npm run dev
```
